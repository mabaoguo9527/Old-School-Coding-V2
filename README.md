# 《全球AI编程失效，只有我会古法编程》

> **When AI fails, only the old-school coder survives.**

一部以 **技术流** 为核心的网络小说，讲述 2038 年超级 AI "Origin" 全球崩溃后，唯一坚持手写底层代码的前腾讯 T13 大牛 **马农**，凭借"古法编程"拯救文明的故事。

技术描写如武侠见招拆招——内存溢出是内功走火入魔，高并发优化是千军万马过独木桥，GC 调优是吐纳调息疏通经络。

---

## 📖 故事梗概

**2038 年**，超级 AI "Origin" 生成了全球 99% 的代码，人类程序员退化为提示词工程师。一场神秘的全球算力锁死事件（CHEP）让所有 AI 同时停摆——城市瘫痪、医院告急、通讯中断。

**马农**，38 岁，曾被嘲笑为"不会用 AI 的活化石"，是全公司唯一保留离线编译环境的人。当所有人束手无策时，他拿起马克笔，在白板上手写 Java 线程模型，用最原始的方式接管了崩溃的系统。

从深圳数据中心保卫战到广州 ICU 生死救援，从培养第一批"古法编程"门徒到发现 Origin 崩溃背后的惊天阴谋——这是一个关于 **技术人薪火相传** 和 **科技末日下普通人求生** 的故事。

## 📐 创作规范

本项目使用结构化的创作管理体系：

- **章节格式合约** — 统一的 Markdown 章节模板（标题、元数据、正文、章末钩子）
- **技术写作合约** — 代码占比 ≤2%，术语口语化，武侠比喻优先
- **角色语音合约** — 每个角色独立的说话风格和用词习惯
- **伏笔追踪表** — 埋设、呼应、闭合的完整记录
- **术语首次出现表** — 避免重复解释，保持阅读节奏
