第2章 乱码

Origin吐出的不是代码，是整个时代的死亡通知。

乱码从屏幕中央开始蔓延，像墨水滴进清水。

一秒前还整齐排列的Java代码变成了无意义的字符洪流——ASCII控制符、半角问号、UTF-8的截断碎片混在一起，以每秒上百行的速度填满六米宽的全息屏幕。

全场的第一反应是笑。

"Origin也有bug？"前排一个候选人推了推旁边的人，"这可是活久见。"

第二秒，笑声还没散尽，评委席上五台笔记本的屏幕同时弹出了警告窗口。

[Origin Alert] Inference latency exceeded threshold: 12ms → 3847ms

三千八百四十七毫秒。从心跳速度的12毫秒直接飙升到将近四秒。不是抖动，是心脏骤停。

老张的笑容凝固在脸上。他这辈子见过很多系统告警，但Origin的告警他只在入职培训的PPT上见过——因为Origin从来没有报过警。

第三秒。

全息屏幕彻底变成了乱码的瀑布。会议室的灯光闪了一下——不是停电，是大楼智能照明系统在重新握手。空调的出风口发出一声细微的咔哒，温度设定从二十三度变成了显示"--"。

候选人张伟还举着手里的翻页笔，嘴巴微张。他不知道该做什么。他的整个职业生涯，从入行到晋升，所有的技术方案都是通过Origin完成的。Origin崩了，等于他的双手被斩断。

第五秒。

有人掏出手机想查Origin的官方状态页。屏幕转了三圈，加载失败。

"网断了？"

不是网断了。是腾讯内网的DNS解析服务——由Origin生成和维护的服务——在失去Origin的在线热修复后，开始返回错误的解析结果。

第十秒。

会议室的门被推开，一个行政助理跑进来，脸色发白："张总，公司系统全崩了。OA、邮件、内部即时通讯……全部503。"

老张猛地站起来。"数据中心呢？"

"正在确认，但是——"助理的声音在发抖，"不只是我们。朋友圈也刷不出来了。微博也上不去。好像……所有的都崩了。"

走廊里响起了脚步声、叫喊声、椅子翻倒的声音。三十三楼的会议室变成了一座信息孤岛——没有网络，没有通讯，甚至没有可靠的照明。应急灯亮了。

恐慌在十五秒内传遍了整层楼。

候选人们有的在反复刷手机，有的在拍桌子大声打电话（无法接通），有的愣在原地一动不动。评委席上，两个人已经冲出去找运维，两个人在对着黑屏的笔记本发呆。

只有老张，双手撑在桌子上，死死盯着全息屏幕上那片乱码的海洋。

他的脑子里翻涌着一个念头：如果Origin真的崩了——不是故障，不是降级，而是完全崩了——他们该怎么办？

他发现自己不知道答案。

因为他不知道公司的系统是怎么运行的。Origin生成了那些代码，Origin维护着那些代码，Origin监控着那些代码。没有人读过那些代码。包括他，一个T11。

然后他想起了一个人。

他转过头，朝最后一排看去。

马农还坐在那里。

不。不对。马农已经不在最后一排了——他不知什么时候走到了全息屏幕旁边。准确地说，是走到了会议室侧面墙壁上的那块白板前面。

白板。

在这个全息投影和Origin主导一切的会议室里，那块两米宽的白板是一件古董——三年前装修时留下的，因为拆掉要填补墙面，就一直挂着当装饰。

马农面对白板站着。背对全场。

他的右手拿着那支从帆布包里抽出来的黑色白板笔。笔帽已经摘了。

全场的混乱在他背后翻涌：有人在喊"打电话给数据中心"，有人在喊"Origin崩了怎么办"，有人在喊"谁能修"。

马农没有回头。

他抬起手。白板笔尖触上白板。

第一行字出现了。安静的、工整的、没有任何多余装饰的手写字——像一个老匠人在木板上刻下的第一刀。

``
Thread State Model:
─────────────────
NEW → RUNNABLE → BLOCKED / WAITING / TIMED_WAITING → TERMINATED
`

有人看到了。混乱中，总有人会被一个静止的背影吸引。

"他在干什么？"

"在白板上……写字？"

马农没有停。第二行出现了。

`
volatile: memory barrier → happens-before guarantee
Write: StoreStore + StoreLoad
Read:  LoadLoad + LoadStore
`

老张从评委席上站起来，穿过混乱的人群，走到马农身后。

他看着白板上的字，心跳忽然漏了一拍。

这是Java内存模型。volatile关键字的内存屏障语义。线程状态机的完整转换图。

没有Origin。没有IDE。没有任何辅助工具。

一个人，一支笔，一块白板。

手写的。

老张张了张嘴，想说点什么。但马农在这时候开口了——不是对他说，是对着白板说，声音不大，但在混乱中异常清晰。

"先稳住线程。"

六个字。

老张愣了一秒。然后他听到马农说了第二句话。

"Origin的推理层锁死了，但JVM还在跑。"

马农转过身。他的表情没有任何波澜——不是装的，是真的没有。就像一个外科医生在急诊室里看到一个大出血的病人，不会先慌，先看伤口。

他用白板笔指了指全息屏幕上的乱码。

"这不是bug。"

全场安静了一瞬。

"Origin没有崩溃。它的推理层进入了一个无限循环。算力被锁死在循环里，进不去也出不来。所有依赖Origin在线推理的代码生成和热维护，全部停止。"

他停顿了一秒。

"但底层JVM进程还活着。堆区在持续分配对象，GC在拼命回收，CPU占用率拉满——因为那个无限循环在疯狂消耗算力。"

他转回白板，写下第三行。

`
Heap: 32GB allocated / 200MB free
GC:   Full GC every 3 seconds (STW)
CPU:  100% (inference loop)
``

"堆区还有200MB。"

他说出这句话的语气，就像说"碗里还有半碗饭"一样平常。

"够了。"

够什么？

老张想问，但马农已经开始在白板上写下一段东西——不是文字，是代码。一行一行的Java代码，工整、简洁，像刻在石碑上的铭文。

白板笔在白板上发出细微的吱呀声。

全场的混乱像退潮一样慢慢安静下来。一个又一个人注意到了白板前那个灰色冲锋衣的背影——他在做一件这个时代没有人做过的事。

手写代码。

会议室里最年轻的候选人——二十三岁，二〇三五年毕业——从未见过有人手写代码。他见过博物馆里展示的"程序员"工位：机械键盘、双显示器、终端窗口里跳动的光标。那是历史。

现在，历史在他面前复活了。

白板上的代码越来越多。线程模型、内存屏障、CAS操作的伪代码——每一行都精确得像是从教科书里撕下来的，又比教科书鲜活一万倍。

因为教科书上的代码是死的。

马农写的代码是用来救命的。

评委席上唯一还坐着的那个人——一个五十多岁的技术顾问——摘下眼镜，揉了揉眼睛，低声说了一句：

"妈的，CAS。我上次见人手写CAS，还是二〇二〇年。"

马农写完了最后一行。转过身。

他看了一眼全息屏幕——乱码还在刷。看了一眼应急灯——走廊里已经全部切换到应急照明。看了一眼手机——黑屏，4G/5G信号全无。

然后他对老张说了第三句话。

"带我去机房。"

老张没有犹豫。他抓起工牌，大步走向门口，回头对会议室里剩下的所有人喊了一句：

"所有人留在原地，不要乱跑。"

然后他看了马农一眼。

马农把白板笔的笔帽盖好，揣回冲锋衣口袋。

两个人一前一后走出会议室。

走廊里一片混乱。有人跑来跑去，有人蹲在墙角刷手机，有人在打不通的电话前反复尝试。应急灯把所有人的影子拉得很长。

马农穿过走廊的时候，走廊安静了一瞬。不是因为人们认出了他，而是因为——在所有人都在慌张的时刻，有一个人走路的步伐和呼吸的节奏，都像平时一样。

他们到了电梯口。电梯停运了。

"走楼梯。"老张说。"机房在地下二层。"

马农点头。

三十五层楼梯。

他们开始往下走。

楼梯间的应急灯每隔两层亮一盏，明暗交替。他们的脚步声在水泥墙壁间回荡。

老张一边走一边说："马农，Origin这次到底怎么了？"

马农的回答只有两个字。

"锁了。"

"谁锁的？"

沉默。三层楼梯的沉默。

然后马农说了一句让老张后来反复回想了很多天的话——

"也许没有人锁。也许它自己锁的。"

老张停住脚步。在应急灯昏黄的光线里，他看到马农的眼睛里没有恐惧，没有兴奋，只有一种他见过无数次但从未在这种场合见过的东西。

专注。

就像盯着一段有bug的代码，准备逐行debug时的那种专注。

但这次要debug的不是一个模块。

是整个世界。
