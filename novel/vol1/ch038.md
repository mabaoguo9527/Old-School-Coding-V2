# 第38章 旁门左道

> JVM字节码本质上也是一种虚拟机指令集。区别只在于——一个跑在硅片上，一个跑在软件上。

---

CHEP后第四十六天。深夜十一点。

设备科。

马农已经读了三个小时的TMS320F28335指令集手册。

八百页。他读到了第四百二十页。不是浏览——是逐行读。每读一条指令，他在笔记本上写一行对照：

```
TI DSP指令 → ARM等价 → Java字节码等价
MOV ACC,loc16 → LDR R0,[R1] → iload
ADD ACC,loc16 → ADD R0,R0,R1 → iadd
```

三列。三种语言。同一个逻辑。

这不是标准做法。没有教科书教这个。但马农不是从教科书学编程的。

他是从师父那里学的。

师父说过一句话："所有指令集长得都不一样，但干的事情就那么几件——搬数据、做运算、跳转。你把这三件事理解透了，什么指令集都是同一种东西。"

马农当时十六岁。觉得师父在说废话。

现在他三十八岁。面前摆着一颗完全陌生的DSP芯片。他发现师父说的是对的。

---

凌晨一点。

赵天来换班——他白天太累，睡了三个小时后来值夜。

他走进设备科，看到马农的屏幕上不是代码，是一张手绘的表格。密密麻麻的十六进制数字。

"你在做什么？"

"翻译。"马农说。

"翻译什么？"

"把C语言翻译成DSP能听懂的话。"

赵天走近了看。马农的笔记本旁边放着一张纸——不是打印的，是手写的。上面画了一个呼吸机控制循环的流程图。每个步骤旁边标注了对应的TI DSP汇编指令和机器码。

"你在手写机器码？"赵天的声音有点不可思议。

"在想另一个办法。"马农说。"手写机器码是最后手段。但有个捷径。"

他把屏幕转向赵天。

上面是一段Java代码。

赵天更困惑了。"Java？不是要写嵌入式固件吗？"

"JVM字节码。"马农说。"和DSP机器码本质上是同一类东西——都是虚拟机指令集。JVM是软件虚拟机。DSP的CPU是硬件虚拟机。它们的指令格式不同，但逻辑结构一样：操作码+操作数。"

他在纸上画了一张图：

```
C代码: x = a + b;
    ↓ (编译器)
DSP机器码: [操作码][地址a][地址b][目标x]
    ↑ (结构相同)
Java字节码: iload_1 / iload_2 / iadd / istore_3
```

"我不需要TI的编译器。"马农说。"我需要理解DSP的指令编码格式。然后我可以——"

他停了一下。

"用Java写一个极简编译器。输入C代码。输出TI DSP的机器码。"

赵天张了张嘴。"你要……用Java写一个C语言编译器？"

"不是完整的C编译器。"马农说。"只处理我固件里用到的那些——变量赋值、算术运算、条件判断、循环、函数调用、中断。没有指针运算。没有结构体嵌套。没有递归。"

他在纸上列了一组核心需求——不到二十种语法结构。

"一个固件用到的C语言子集，翻译成DSP机器码。用Java跑在我的笔记本上。"

---

这个思路不是凭空冒出来的。

马农在腾讯T11的时候，做过一件事：为了排查一个JIT编译器的bug，他手动把一段Java字节码反编译成汇编，和HotSpot生成的本地代码逐行对比。那次他花了一周，理解了JVM字节码到x86机器码的完整翻译过程。

那个经验现在救了他。

因为从"高级语言到机器码"的翻译，不管目标平台是x86、ARM还是TI DSP——步骤都是一样的：词法分析→语法分析→中间表示→目标代码生成。

区别只在于最后一步：生成什么样的机器码。

而TI DSP的指令编码格式，在手册的第五章到第八章有完整定义。每条指令几个字节、操作码在哪些位、操作数怎么编码——全部写得清清楚楚。

他只需要把这些编码规则翻译成Java代码。

"旁门左道。"马农自己说了四个字。

赵天没听清。"什么？"

"师父以前说——正道是拿对的工具做对的事。旁门左道是——手边没有对的工具，用错的工具做对的事。"

他打开了一个新的Java文件：`TinyC2000Compiler.java`。

开始写。

---

凌晨两点到五点。

马农写了三个小时。

这个"极简编译器"分三层：

第一层：词法分析。把C代码切成一个个记号——变量名、数字、运算符、关键字。这一层他闭着眼都能写——和写一个JSON解析器的难度差不多。

第二层：语法分析。把记号组合成语法树——赋值语句、if-else分支、while循环、函数定义。他只处理呼吸机固件里用到的结构，忽略所有复杂特性。不支持指针算术。不支持多文件编译。不支持预处理器。

第三层：代码生成。遍历语法树，为每个节点生成对应的TMS320F28335机器码。这一层最难——他需要把手册里的指令编码规则逐条实现。

`MOV ACC, #imm16`——把立即数加载到累加器。操作码`0x9A00`，后跟16位立即数。

`ADD ACC, loc16`——把内存值加到累加器。操作码取决于寻址模式——直接寻址`0x0600`，间接寻址`0x8200`。

一条一条。

赵天在旁边看了一会儿。他看不懂指令编码的细节，但他看懂了结构——马农在用Java的`HashMap`建了一张指令编码表，键是助记符，值是操作码和格式描述。然后用一个`switch-case`根据语法树节点类型分发到不同的编码函数。

"这个结构……"赵天说。"有点像我们在903写的路由分发？"

马农停了一下。他看了赵天一眼。

"本质上是一样的。"他说。"路由分发是根据URL映射到处理函数。代码生成是根据语法节点映射到机器指令。模式相同。"

赵天忽然觉得自己好像理解了一点什么。不是技术上的理解——是一种更深的直觉：**编程世界里的所有东西，在足够高的抽象层面上，都是同一件事。**

---

早上六点。

赵天出发去广州生物医学工程研究中心找TI编译器。带着小陈。

"如果找到了，U盘拷回来。"马农说。"如果没找到——不要浪费时间。一个小时内回来。"

赵天点头出发了。

马农继续写他的编译器。

八点。词法分析完成。
九点。语法分析完成。
十点。代码生成——完成了算术运算和变量赋值。

还差循环、条件分支、函数调用、中断处理。

十点半。赵天回来了。

"研究中心的电脑全部联网的。开不了机。"赵天说。"但是——"

他从口袋里掏出一个U盘。

"实验室的一台旧电脑。XP系统。离线的。开机了。上面装着Code Composer Studio v5。我把整个安装目录拷下来了。"

马农接过U盘。

他插进笔记本电脑。打开文件夹。找到了`cl2000.exe`——TI C2000系列的命令行编译器。

Windows的。

马农的笔记本是Linux。

他试着用Wine运行。失败——缺依赖库。

他试着找Linux版本。U盘里没有。

他看了一眼自己写了一半的Java极简编译器。

然后他看了一眼U盘里Code Composer Studio的安装目录。里面有一个文件夹叫`include`——TI DSP的头文件。另一个文件夹叫`lib`——链接库。还有一个文件夹叫`doc`——完整的编译器手册。

"赵天。"马农说。

"在。"

"这个编译器我用不了。但它的头文件和文档——能用。"

他打开了`include`目录。里面有TMS320F28335的全部寄存器定义——每个外设的地址、每个位域的含义。这些信息比手册更精确，因为是直接从芯片厂商的SDK里提取的。

他把这些头文件的信息整合进自己的Java编译器——不是复制粘贴，是提取关键的寄存器地址和位定义，硬编码进代码生成器。

十一点。继续写代码生成层。

下午一点。循环和条件分支完成。

下午三点。函数调用和中断向量表完成。

下午四点。

马农把呼吸机固件的C代码喂进了自己写的`TinyC2000Compiler`。

敲下回车。

终端停了两秒。然后输出了一个二进制文件：`ventilator_pb840.bin`。

182KB。

他用十六进制编辑器打开。看了前几十个字节——中断向量表、复位入口地址、栈指针初始化。格式和手册里描述的TMS320F28335的Flash布局吻合。

"赵天。把PB840的JTAG接好。"

---

下午四点半。

Medtronic PB840。最后一台。

JTAG接口和STM32不同——TI用的是14针的JTAG标准。小陈翻了半天工具箱，找到了一个万用转接头。接好了。

马农先备份原始Flash。然后擦除。然后写入。

十四秒。

上电。

LCD黑了两秒。然后亮了。自检信息滚过——不是Origin的界面，是马农固件的极简文本。

传感器校准——这次他提前查了PB840的传感器型号和灵敏度系数，写进了固件里。

步进电机启动。

串口日志：压力曲线。

20cmH2O。稳定。没有偏移。

赵天盯着屏幕。三十秒。一分钟。两分钟。

曲线平滑得像一条直线。

"通过。"赵天说。

马农没有说话。

他跑了完整的异常测试。传感器断开。过压。欠压。断电恢复。参数边界值。

全部通过。

---

下午五点。

马农走到ICU门口。李敏在里面。

"最后一台。"马农说。"2号床。准备手动气囊团队。"

李敏看了他一眼。她的眼睛里有一种马农不常看到的东西——不是感激，不是敬畏。是累了太久之后终于看到终点的那种表情。

她转身走进ICU。

五分钟后。

手动气囊团队就位。2号床旁边。两个人。

2号床的病人——COPD急性发作。持续通气四十五天。

马农蹲下来。JTAG已经预连接好了。

"准备。"

"断电。"

小陈按下开关。呼吸机灭了。手动团队立刻接管——气囊扣上面罩。一按。松。一按。松。

马农的手指落在键盘上。

擦除。写入。校验。

"十五秒。"赵天报时。

上电。

呼吸机亮了。自检。校准。启动。

压力曲线跳出来。稳定。

"十八秒。"

马农站起来。

"十一台。全部完成。"

ICU里安静了一秒。然后有人在鼓掌。不知道是谁先拍的。掌声稀稀落落的——因为大部分人的手都在抖，拍不响。

但掌声持续了很久。

---

那天晚上。

护士站。

李敏把那些手动气囊收进了柜子里。蓝色的橡胶已经被握得变了色——深色的、浅色的、有裂纹的。每一个上面都有手指的形状。

她关上柜门的时候，手又抖了。

但这次不是因为累。
