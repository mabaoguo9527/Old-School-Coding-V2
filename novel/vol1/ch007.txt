第7章 四十八小时

预言不需要超能力，只需要看清依赖链。

CHEP后第六小时。

深圳的夜晚第一次回到了真正的黑暗。

不是那种"停电了点蜡烛"的黑——城市电网还在运转，至少暂时还在。但所有由Origin控制的智能照明系统都停了。路灯只有最基础的那批——十年前装的钠灯——还亮着，发出暗淡的橘黄色光。新装的LED智慧灯杆全部黑了。

从903室的窗口望出去，深圳的天际线像一张被PS去掉了发光图层的照片。轮廓还在，但没有了灵魂。

马农没在看窗外。

他已经连续写了四个小时的代码。SRV-1的终端上开着四个tmux窗格：一个在编译，一个在跑测试，一个在监控JVM状态，一个在写下一段代码。

手摇收音机开着，音量很低。深圳交通广播已经变成了滚动应急播报，每十五分钟重复一次。

马农一边写代码一边用耳朵过滤信息。

"……市区主要路口已部署交警人工指挥，市民出行请注意安全……"

第一批——交通。已倒。

"……银行系统临时关闭线上服务，各网点明日起提供现金窗口服务，请市民携带身份证件……"

第二批——金融。正在倒。比他预估的快了两小时。

"……三大运营商声明：受Origin系统故障影响，4G/5G服务出现大面积降级。部分地区2G语音通话仍可使用……"

第三批——通讯。正在倒。

马农的手指在键盘上停了一秒。

2G。

他从抽屉里翻出一部老旧的诺基亚——2G功能机，插着一张多年未用的联通卡。开机。屏幕亮了，绿色的，像一扇通往2010年的窗户。

信号栏：两格。

他拨了一个号码。嘟——嘟——嘟——

"喂？"老张的声音，背景噪音很大。

"2G还能用。"马农说。

"什么？"

"2G基站的核心协议栈是硬编码在基带芯片里的，不走Origin。保存好你能找到的所有2G手机。"

"马农，现在公司这边——"

"说重点。"

"CTO刚开了紧急会。全公司能手动操作服务器的人——不是写提示词，是真能敲命令行的——凑了不到二十个。其中能写Java的，七个。能写不依赖Origin的Java的——"

"几个？"

"CTO说，可能就你一个。"

马农没说话。

"服务器的事我在推。后勤已经从福田仓库找到了十二台旧服务器，明天一早用卡车运过来。发电机还在协调。"

"UPS能撑多久？"

"数据中心的UPS加柴油发电机，理论上七十二小时。但柴油储备只够四十八小时满负荷运转。"

四十八小时。又是四十八小时。

"知道了。"马农挂了。

他回到终端，打开之前写的应急行动清单，加了一行：

``
P-1 — 生死线
0. 数据中心柴油储备：48h。48h后断电 = 所有服务器停机。
   一切工作必须在48h内完成到可离线运行的状态。
`

CHEP后第十二小时。凌晨两点。

马农完成了应急路由服务的核心代码——EmergencyRouter，两百四十行Java。纯Socket编程，不依赖任何框架，不依赖Origin，不依赖互联网。

`java
// EmergencyRouter v0.1 — 核心流量分发
// 依赖: JDK 17 only. 零外部依赖.
// 编译: javac EmergencyRouter.java
// 运行: java EmergencyRouter 8080

public class EmergencyRouter {
    private final int port;
    private final Map<String, String> routeTable;
    private final ExecutorService pool;
    
    // 路由表从本地文件加载，不走任何远程配置中心
    public EmergencyRouter(int port) {
        this.port = port;
        this.routeTable = loadRoutes("/etc/emergency/routes.conf");
        this.pool = Executors.newFixedThreadPool(
            Runtime.getRuntime().availableProcessors() * 2
        );
    }
    // ...
}
`

他把代码编译了一遍。零错误。

然后在SRV-1上启动了一个实例。端口8080。

`bash
java -Xmx512m -XX:+UseG1GC EmergencyRouter 8080
`

`
[EmergencyRouter] Started on port 8080. Routes: 23 services mapped.
[EmergencyRouter] Memory: 512MB max, G1GC, no Origin dependency.
[EmergencyRouter] Status: READY.
`

一个不依赖Origin的路由服务，在一台不联网的物理服务器上活了过来。

马农看了一眼。没有任何庆祝的表情。就像一个木匠钉完了一颗钉子——活儿该这样，没什么好高兴的。

他开始写下一个模块：EmergencyDNS。

一个迷你的DNS服务器。不走根域名系统——根域名系统的智能调度层被Origin控制，现在也是瘫的。这个DNS只负责一件事：把腾讯内部的域名解析到物理IP地址。

`java
// 最朴素的DNS——一个HashMap
Map<String, String> dnsTable = new HashMap<>();
dnsTable.put("core.tencent.internal", "192.168.1.11");
dnsTable.put("auth.tencent.internal", "192.168.1.12");
dnsTable.put("monitor.tencent.internal", "192.168.1.13");
`

他一边写，一边在收音机里听到了第一条真正让他停下来的消息。

"……南方电网通报：由于智能调度系统故障，深圳宝安区和光明区发生计划外停电。电网正在切换到手动调度模式，预计四到六小时内恢复。请受影响区域居民注意用电安全……"

停电。

马农放下手。看了一眼手腕上的卡西欧电子表。凌晨2:37。

第四批——电网。开始倒了。

他预估的时间线是"四十八小时后回到1990年"。但电网的倒塌比他预估的快。

原因也不难推测：电网的负荷调度系统对实时性的要求极高。发电量和用电量必须每时每刻保持平衡——多了要降出力，少了要加出力。这个平衡由Origin管理的智能调度算法维持。Origin死了，算法死了，平衡就开始失控。

变电站的保护继电器检测到频率偏差，自动切断了部分区域的供电。这是保护机制——宁可局部断电，也不让整个电网崩溃。

但这只是开始。

马农重新打开应急清单，把电网的优先级从P1提升到了P0。

`
P0 — 立即
1. 腾讯数据中心核心服务手动维护方案 [进行中]
2. 内网通讯恢复 [进行中]
3. 数据中心供电保障（柴油48h → 需要外部供油）
4.  城市电网状态 → 已开始局部停电 
`

CHEP后第二十四小时。下午两点。

马农没有睡。

不是因为紧张。是因为没有时间。

十二台旧服务器在上午被一辆改装过的卡车运到了B栋楼下——卡车是手动挡的老款五十铃，司机是腾讯后勤部一个五十多岁的老师傅。这位老师傅是整个后勤部唯一一个还会开手动挡卡车的人。

马农和老张带着七个"能敲命令行"的腾讯工程师，把十二台服务器搬上了九楼。没有电梯。十二趟楼梯。每台服务器四十斤。

七个工程师搬完服务器之后瘫坐在走廊上喘气。马农没歇。他开始接线。

网线。电源线。串口线。

他把十二台服务器加上原有的三台，用交换机组成了一个十五台的局域网集群。手动配IP。手动设路由表。手动写/etc/hosts。

没有DHCP——DHCP服务器是Origin管理的。  
没有DNS——他自己写的EmergencyDNS刚刚上线。  
没有负载均衡——后面再说。  
没有监控——后面再说。  

先把基础服务跑起来。

下午一点，他在十五台服务器的集群上启动了三个核心服务：EmergencyRouter（路由）、EmergencyDNS（域名解析）、EmergencyMonitor（基础监控——每30秒采集一次CPU/内存/磁盘，写到本地日志文件）。

"丑陋。"马农看着监控的输出。纯文本的。没有图表，没有仪表盘，没有Origin那种华丽的可视化。

但能用。

他转过身，面对那七个工程师。

这七个人是CTO从全公司两万多名员工里筛出来的——"能敲命令行"的标准线。其中最年轻的二十四岁，最老的四十一岁。四个后端，两个运维，一个做了十年DBA但五年没碰过SQL了（因为Origin自动生成SQL）。

他们站在一排服务器前面，像站在博物馆的展品前。

马农扫了他们一眼。

"你们之中，有人能不用Origin写一个HTTP服务器吗？"

沉默。

"Socket API？ServerSocket？HttpURLConnection？"

更深的沉默。

"有人知道TCP三次握手的SYN和ACK分别是什么吗？"

一个四十一岁的后端工程师举了手。犹豫地。

"SYN是同步序列号……ACK是确认应答……"

"然后呢？"

"……然后Origin会自动处理。"

马农没有表情。

他走到白板前。擦掉了上面的Linux CFS红黑树图（他在心里和它道了个别），拿起马克笔。

在白板正中央写了一行字。

`
第一课：for循环
`

七个人面面相觑。

最年轻的那个——二十四岁的后端工程师——忍不住开口了。"马……马农哥，我们都会for循环啊……"

"写一个。"

"什么？"

"在白板上。写一个for循环。不用伪代码。Java。from memory。"

年轻人犹豫了三秒。走到白板前。拿起笔。

写了一行。

`java
for (int i = 0; i < 10; i++)
`

然后停了。

"括号后面呢？"马农问。

"……大括号。"

"写。"

年轻人写了大括号。里面是空的。

"你打算让它循环什么？"

"呃……System.out.println……"

"写。"

年轻人写完了。一个打印0到9的for循环。基础得不能再基础。

马农看了五秒。

"i。"他说。

"什么？"

"变量名。i代表什么？"

"循环变量……"

"为什么叫i？"

"因为……所有人都叫i？"

马农的表情终于有了一丝变化。不是怒——是一种介于叹息和无奈之间的东西。

"i是index的缩写。"他说。"但你在这里循环的是计数，不是索引。"

他拿过笔，在年轻人写的代码旁边写了一行：

`java
for (int count = 0; count < 10; count++) {
    System.out.println("Attempt " + count);
}
`

"变量名是给人看的。"

七个人安静了。

马农把笔帽盖好。

"四十八小时。"他说。"这是城市的时间。也是你们的时间。四十八小时之内，你们要从'会写提示词'变成'能维护一个基础Java服务'。"

他看了一圈他们的脸。

"我不期望你们写出漂亮的代码。能跑、不崩、可以debug——就够了。"

他顿了一下。

"先活着。然后再谈别的。"

他转回终端。手指落在键盘上。

屏幕上的EmergencyMonitor刷出了最新一条日志：

`
[2038-09-14 14:12:33] [WARN] SRV-7: disk usage 89%. 
Reason: Origin cache files not cleaned. 
Action required: manual cleanup.
``

马农叹了口气——不，他不叹气。他只是呼吸节奏变了一下。

Origin留下的缓存文件在吃磁盘空间。还是Origin的烂摊子。

他回头看了一眼那七个人。他们还站在白板前面，有人在偷偷拿手机拍那段for循环的代码——用的是2G手机的低像素摄像头。

城市在黑暗中等待。

而他要在四十八小时内，把七个只会写提示词的人变成最后一道防线。

窗外，收音机里传来第四十七次的循环播报。

"……请市民保持冷静。Origin系统正在抢修中……"

马农没关收音机。他需要它来校准时间——每十五分钟的播报间隔是他唯一的外部时钟。

他的手腕上，卡西欧电子表显示：14:13:07。

CHEP后第二十四小时零十三分。

还剩二十四小时。
