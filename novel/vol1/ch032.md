# 第32章 信使

> 有些求助不是通过网络来的。是用脚走来的。

---

CHEP后第四十天。

广州的消息越来越急。

```
[接收] 2038-10-15 11:03 | 广州天河
柴油只够三天了。发电机停了就全完了。
另外：市三院联系到我们了。他们ICU有11台呼吸机。
固件全部崩溃。手动维持呼吸已经三周了。
护士们轮班手捏气囊。24小时不停。
她们快撑不住了。
你们……有人能来吗？
```

马农读了两遍。

然后他把这条消息打印出来——903室的打印机是他从数据中心搬来的，连着一台老式激光打印机。

他把纸贴在了白板上。

---

但在他回复之前，另一件事发生了。

下午两点。903室的门被敲响了。

开门的是老李。门外站着一个人——二十出头，穿着一件脏兮兮的冲锋衣，背着一个登山包。脸被晒得通红。鞋底磨平了。

他说："请问……马农在吗？"

"你是谁？"老李问。

那个人从登山包里掏出一个塑料袋。塑料袋裹了三层，最里面是一张折叠的纸。

纸上写着：

```
致深圳数据中心903室：
我是广州市第三人民医院ICU护士长。
我们的呼吸机已经全部崩溃。11台。
手动维持呼吸42天了。
我们听说903室有人会修代码。
这个年轻人叫小周。他志愿走过来的。
走了六天。
请帮帮我们。
```

信的右下角有一个红色的印章。广州市第三人民医院。

马农看着这个叫小周的年轻人。

"你走了六天？"

"四天走到东莞。在东莞搭了一段车。最后两天从东莞走到这里。"

"你怎么知道903室？"

"我们的通讯系统收到了你们的信号。"小周说。"天河移动机房那边的人告诉我们——深圳有个叫马农的人，会写代码。"

马农看着信纸上的字。护士长的字很工整，但有几处墨迹晕开了——可能是写的时候手在抖。

他把信纸贴在了白板上。就贴在广州消息的旁边。

---

903室临时开了一个会。

与会者：马农、老李、赵刚、刘韧、林夕。

议题：是否赴广州。

赵刚先说了物理条件："路是通的。广深高速的东莞段有坍塌，但可以绕行。柴油越野车能跑。单程大概需要两天——不能走高速的话得走省道。"

老李说了系统情况："903的三个核心服务都在稳定运行。我可以独立维护一到两周。但如果出了重大故障……"

"刘韧可以帮你。"马农说。

刘韧点头："我对路由服务已经能上手了。DNS和认证服务——我需要看一遍马农的代码再说。"

"你有三天时间看。"马农说。"出发前我会给你做交接。"

林夕一直没说话。

"林夕。"马农看着她。

"嗯。"

"你留在这里。继续教第二批学员。手册你有了。课程表你有了。老李和刘韧负责系统。你负责教。"

林夕的嘴唇动了一下。想说什么。没说出来。

"你说。"马农说。

"我想跟你去。"林夕说。"广州的呼吸机——那是嵌入式系统。ARM架构。你一个人——"

"我不是一个人。"马农说。"我带赵天和小陈。"

"赵天？"林夕有点意外。赵天才学了四周。

"赵天需要实战。他的问题不是能力——是没见过真正的代码战场。"马农说。"小陈熟悉硬件。赵天管软件端。我做架构和兜底。三个人够了。"

他看着林夕。

"你在这里比在广州更重要。"

林夕低下头。

"你不只是留下来看家。你是在建下一代。"马农的声音很平。但每个字都有重量。"如果我在广州出了什么事——903的未来在你手上。不是因为你技术最强——是因为你理解为什么要做这件事。"

林夕的眼睛红了。但她没有哭。

"好。"她说。"我留下。"

---

马农用了三天做准备。

第一天：系统交接。他把所有服务的代码、配置、运维手册整理成文档，逐条讲给刘韧和老李。

第二天：装备准备。他列了一张清单：

```
赴广州装备清单：
1. 笔记本电脑×2（预装JDK 17 + GCC + arm-none-eabi-gcc）
2. 串口线×3 + USB转串口适配器×2
3. JDK离线安装U盘×5
4. 古法编程手册v1.1 U盘×10
5. 工具箱：示波器（手持）、万用表、焊接工具
6. 食物和水：5天量
7. 柴油：备用20升
8. 通讯设备：对讲机（与903保持联系）
```

第三天：给林夕做最后的教学交接。

他们在903室的白板前站了两个小时。马农把第二批培训的注意事项一条一条讲。

"第一周不要急。重复第一批的内容。但你可以加自己的理解。"

"第二周开始教面向对象。如果有人跟不上——不要等他。让他重学第一周。不是所有人都能跟上的。"

"教学的核心不是让他们记住API。是让他们理解逻辑。"

"如果有人问你不会的问题——不要猜。说'我不知道，我去查'。这比给错误答案好一百倍。"

林夕在笔记本上记。

最后一条。

"如果超过七天我没有联系——"马农停了一下。"联系CTO。让他派人来接管903室。不要等我。"

"七天。"林夕重复了一遍。

"七天。"

---

出发那天早上。

赵刚开车。马农坐副驾驶。赵天和小陈坐后排，中间夹着两台笔记本电脑和一个工具箱。

903室的门口。林夕、刘韧、老李、还有八个编程班学员站在那里。

马农拉开车门。然后停了一下。

他回头看了一眼903室。

楼顶的天线。窗户里的灯光。白板上还没擦掉的代码。

他上了车。

赵刚发动引擎。柴油发动机的声音在空旷的科苑路上回荡。

林夕站在门口。她没有挥手。

她低头看了一眼自己的手。手里握着那个U盘——三年前马农备在903室的JDK离线安装包。现在这个U盘是她的了。

车开走了。

903室的灯还亮着。
