第14章 第一盏灯

黑暗中点亮的第一盏灯，照的不是路，是人心。

CHEP后第六十五小时。

深圳市南山区的区政府发出了第一道不依赖任何电子系统的公告。

一张A3白纸，用粗体黑色马克笔手写。贴在区政府大楼正门。内容如下：

``
南山区应急公告（第三号）
2038年9月16日

一、全区电力将优先保障医院、自来水厂、政府应急中心
二、科技园B栋903室已建立临时通讯节点，可提供：
    - 局域网接入（仅限南山区范围）
    - 文字消息传输（类似短信）
    - 紧急联络转接
三、有通讯需求的市民请前往科技园B栋一楼登记
四、请保持秩序。水和食物将在各社区发放点统一分配
`

公告的第二条提到了903室。

马农不知道这个公告是怎么出来的。他猜是老张。老张昨天又打了电话来——2G，断断续续——说了一句"CTO想让市里知道你们那边能通讯"。马农说"随便"，然后挂了。

他不在乎谁知道。他在乎的是：如果有人接入，服务器扛不扛得住。

结果比他预想的快。

上午十点，B栋一楼大厅开始排队。

马农从窗户看下去的时候吓了一跳——不是被人数吓到，而是被队伍的长度。从B栋大门一直排到了科苑路上。至少两百人。可能三百。

他们带着各种各样的东西：笔记本电脑、平板、甚至几台台式机显示器——后者显然没用，但带来的人可能不知道"没有网络"和"没有显示器"是两件事。

赵刚跑上来报告："楼下来了一帮人，说要联网。"

"他们怎么联？"

"我不知道。有个人拿着一根网线问我插哪里。"

马农想了三秒。

903室的临时局域网目前只有十五台服务器和八个人在用。网络拓扑是一条直线——所有设备接在同一台交换机上。如果突然涌入三百台设备，交换机会瞬间过载，ARP广播风暴会把整个网络拖垮。

"不能直接接。"马农说。"需要一个接入层。"

他走到白板前。

`
接入架构：
                    903室（核心层）
                        │
                 ┌──────┴──────┐
                 │  L3 Router  │  ← 马农的EmergencyRouter
                 └──────┬──────┘
                        │
            ┌───────────┼───────────┐
            │           │           │
        ┌───┴───┐  ┌───┴───┐  ┌───┴───┐
        │ AP-1  │  │ AP-2  │  │ AP-3  │  ← 接入点（每层一个）
        └───┬───┘  └───┬───┘  └───┬───┘
            │           │           │
        [用户×50]  [用户×50]  [用户×50]
`

三层架构。核心层是903室的服务器集群。中间是路由器做流量控制。底下是接入点——每层楼放一台交换机做用户接入，每个接入点限制五十台设备。

"交换机哪来？"老李问。

"数据中心搬不了了。"马农说。"但B栋每层楼的弱电间都有网管交换机——物业用来给每层办公室接网的。Origin停了以后那些交换机也闲着。我们用它们。"

"网线呢？B栋内部的布线走的是Origin管理的智能网管——"

"拔掉智能网管模块，回退到普通二层交换。"马农说。"二层交换不需要Origin。以太网帧的转发逻辑烧在ASIC芯片里，纯硬件实现。"

老李愣了一下。然后缓缓点头。"对……交换机的基础转发是硬件做的，Origin只管了VLAN和QoS的软件策略。"

"把VLAN和QoS全关掉。裸跑。"

两个小时后，B栋的一楼、三楼、七楼各部署了一个接入点。

马农没有亲自去布线——他让赵刚和老李带着两个工程师干。他自己在903室做另一件事：给EmergencyRouter加了一个限流模块。

不是什么高级的令牌桶或漏桶算法。是最简单粗暴的计数器限流：

每个接入点每秒最多处理100个请求。超过的直接返回503。

马农在白板上写给团队看："限流不是为了刁难用户，是为了保护服务器。一台服务器能扛的并发量是有限的。超载了所有人都没得用。限流是让100个人慢点用，好过让1000个人一起等着用不了。"

下午一点。第一批用户接入。

马农在监控面板上看着连接数从零开始跳动。1……5……12……27……

每一个数字代表一台设备成功连入了局域网。

他给用户提供的不是"互联网"——互联网还不存在。他提供的是一个局域网消息系统——类似二十年前的BBS。用户用浏览器访问一个内网IP，进入一个极简的Web页面。页面上只有两个功能：

1. 发消息：输入收件人用户名和文字内容，点击发送。消息存在服务器上。
2. 收消息：刷新页面查看是否有新消息。

没有图片。没有视频。没有表情包。只有文字。

那个Web页面是马农昨晚用纯HTML+Servlet写的。不到一百行。界面丑得像1995年的雅虎——白底、黑字、蓝色超链接、Times New Roman字体。

但它能用。

下午三点。连接数到了89。

马农看到了第一条用户发送的消息。

`
发件人: user_023 (张明)
收件人: user_067 (李芳)
时间: 2038-09-16 15:03:22
内容: 老婆，我在科技园B栋。我没事。你和孩子在家等着，别出门。
      水和食物够吗？我明天想办法回来。
`

马农的目光在这条消息上停了两秒。

然后他翻到了下一条。

`
发件人: user_045 (王磊)
收件人: user_045 (王磊)  ← 给自己发
时间: 2038-09-16 15:07:11
内容: 测试。这真的能发出去吗？我是在做梦吗？
      三天了。三天没联系上任何人。手机就是一块砖。
      如果有人看到这条消息，请告诉我我不是一个人。
`

马农关掉了消息列表。

他不看用户的私人消息。那是他写的认证系统应该保护的东西。他只是在测试的时候瞥到了。

但那两条消息让他意识到一件事：他搭建的这个丑陋的、简陋的、用vi写出来的局域网消息系统——对这些人来说，是三天里第一次和另一个人类产生文字连接。

比技术重要的，是连接本身。

傍晚五点。连接数到了143。

一楼大厅还在排队。赵刚下去维持秩序。有些人没有笔记本电脑，问能不能借一台。赵刚找了几台办公室里的废弃台式机——Origin停了以后它们就没人用了——擦干净接上网线，放在一楼大厅当"公用终端"。

马农在监控面板上看到了一个熟悉的模式：连接数在匀速增长，但消息发送的峰值集中在刚接入的前五分钟。每个人连上以后做的第一件事都是发消息——找家人、找朋友、找同事。五分钟后，大部分人发完了，就安静了。

流量模式和他预期的一样。不是持续高并发，是脉冲式。

他微调了限流参数。突发允许更高，但持续流量收紧。

六点。一个意外发生了。

老李跑过来："马农，一楼有个女的非要上来找你。"

"谁？"

"不认识。说她是腾讯的，T7，想学——"老李顿了一下。"想学写代码。"

马农没说话。

"保安拦住了。她在楼梯口等着。要不要见？"

马农站起来。走到903室门口。打开门。

楼梯口站着一个年轻女人。二十五六岁的样子。高马尾。穿着一件皱巴巴的白色T恤和牛仔裤——大概三天没换了。手里拎着一个帆布包，包上印着"2035 CSW Challenge"的字样——某个编程马拉松的周边。

她看到马农的时候，站直了。

"你是马农？"

"嗯。"

"我叫林夕。腾讯T7，提示词工程师。"她深吸一口气。"CHEP那天我在数据中心做前端部署验收。系统崩了以后我被困了一天半。后来有人用钥匙开了门——我听说是你让保安开的。"

马农记得。他到数据中心那天确实让保安开了几个区域的门。

"我出来以后走到了科技园。"林夕说。"听说这栋楼有网络。我排了四个小时的队。发了一条消息给我爸妈——他们在杭州，不知道收不收得到。"

马农没接话。

"然后我看到了你写的那个消息系统的页面。"林夕说。她的声音开始发抖——不是害怕，是某种被压了很久的东西在往外涌。"HTML。纯HTML。没有CSS框架，没有React，没有Origin。就是最原始的那种。"

"我在大学里学的是前端。"她说。"但我从来没写过一行HTML。都是Origin帮我写的。那天我看到你的页面——那个丑得不行的页面——我才知道……"

她停了一下。

"我才知道，我什么都不会。"

马农看着她。

沉默了五秒。

"你想学什么？"

"我想学……从第一行开始。"她说。"教我写代码。真正的代码。不是提示词。"

马农看了她三秒。

然后他转身走回903室。走了两步，停下来，没有回头。

"明天早上八点。自己带纸和笔。"

他关上了门。

那天晚上，马农坐在SRV-1前，看着监控面板。

连接数稳定在168。消息系统运行平稳。没有告警。GC停顿在200毫秒以内。路由无丢包。DNS解析延迟在3毫秒。

一切正常。

他打开了消息系统的后台统计——不是看内容，是看数据：

`
[Message Statistics] 2038-09-16 23:59:59
Total users registered: 168
Total messages sent: 2,347
Average messages per user: 13.97
Peak send rate: 47 msg/min (at 15:05)
Current send rate: 3 msg/min
``

2347条消息。168个人。平均每人发了14条。

马农关掉了统计。

他走到窗前。深圳的夜空依然没有灯光——除了远处零星的几处火光。可能是有人在烧柴取暖，也可能是别的什么。

903室窗户的灯光从九楼投射出去。在一片漆黑中，它是可以看到的。

从楼下看上去，那大概就是——

第一盏灯。
