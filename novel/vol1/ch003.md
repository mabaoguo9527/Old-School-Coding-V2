# 第3章 马克笔与白板

> 他用一支笔，让整个时代闭嘴。

---

地下二层机房的门是一道重达两百公斤的防火密闭门，需要工牌+虹膜双重验证。

虹膜系统在三秒前挂了。

老张刷了五次工牌，红灯。门禁系统连着腾讯内网认证服务——Origin维护的那套微服务。Origin锁死，认证服务自然断链。

"备用方案呢？"马农问。

老张愣了一秒。他是T11，但他从未想过这道门打不开的情况。这道门从来没有打不开过。

"有……物理钥匙。但在行政部保险柜里。行政部在十五楼。"

马农看了一眼门框侧面的应急面板。灰色铁盒，"EMERGENCY"字样，一把红色的机械旋钮。

"这个。"

"那是消防应急开关——"

马农已经拧了。

液压锁释放。密闭门无声滑开。

机房内部的温度比走廊低十度。冷通道的蓝色LED灯带在黑暗中勾勒出一排排机柜的轮廓——四百多台服务器，风扇声汇成低沉的轰鸣，像一头沉睡巨兽的呼吸。

但呼吸不对劲。

马农站在机房入口没动。他在听。

普通人听到的是一片嗡嗡噪声。但马农听到了节奏。机房风扇的转速由温控系统自动调节——服务器负载高，发热大，风扇转速快，音调高；负载低，音调低。正常运行时，四百台服务器的风扇声会形成一种有规律的波动，像潮汐。

现在没有波动。所有风扇都在满速运转。

"全部拉满。"马农说。

"什么意思？"老张跟在后面。

"每台机器的CPU都跑满了。"马农走向最近的一排机柜。"Origin的推理循环在吃算力。"

他走到第一个机柜前，拉开前面板。一台2U服务器，状态灯全红。

"有KVM吗？"

"第三排末端有一台紧急运维终端。"

马农穿过冷通道。脚步声被风扇噪音淹没。他找到了那台终端——一个十五寸的液晶屏，连着一套键鼠，通过KVM切换器可以接入机房内任意一台服务器。

屏幕是亮的。上面显示着一个登录界面——但不是Origin的管理面板，而是一个原始的Linux TTY终端。

`login: _`

马农坐下来。

他的手指落在键盘上。停了一秒。

这一秒不是犹豫。是确认。确认键盘布局、键程、回弹力度。就像一个剑客拔剑前先感受剑柄的重量。

`root`

回车。

密码？他看了老张一眼。

老张报了一串字符。马农没让他重复，一次输入。

终端闪了一下。进去了。

```
[root@tn-shenzhen-core-001 ~]# 
```

马农开始敲命令——`top -H -p`查看进程线程。屏幕刷出了一片进程列表，两百多个线程，每一个的CPU占用都在99%以上。

老张凑过来看，倒吸一口气："所有推理线程都在空转？"

马农没回答，已经在敲下一条命令——jstack导出线程堆栈，head查看前100行。屏幕上涌出了密密麻麻的线程堆栈信息。对大多数人来说，这是一堆无法解读的天书。但马农的眼睛在上面扫过，速度快得像在读一份菜单。

他停住了——所有推理线程都卡在RUNNABLE状态，不是BLOCKED，不是WAITING，是RUNNABLE——它们认为自己在正常工作。

老张不明白。

"所有推理线程都卡在RUNNABLE状态。"马农指着屏幕。"不是BLOCKED，不是WAITING。是RUNNABLE——它们认为自己在正常工作。"

他站起来。

转过身，面对老张。

"线程状态模型。"他说。"你知道Java线程有几种状态吗？"

老张张了张嘴。他知道——不，他曾经知道。那是十五年前面试时背过的知识点。

马农没等他回答。他从冲锋衣口袋里抽出了那支黑色马克笔。

机房的墙壁上有一块白板。运维人员平时在上面记录巡检信息的那种白板，上面还残留着几行潦草的日期和机柜编号。

马农走过去，用手掌抹掉了旧字迹。

笔帽落在地上。他没捡。

第一行字出现在白板上——Java线程六状态模型：NEW → RUNNABLE → {BLOCKED | WAITING | TIMED_WAITING} → TERMINATED。他的字很工整，不是艺术字的工整，是工程图纸的工整——每个字母间距均匀，箭头方向精确。

"六种状态。"马农的声音平稳得像在念说明书。"一个线程从NEW到TERMINATED，中间必须经过RUNNABLE。"

他在白板上画了一个圆圈，标注"RUNNABLE"。

"正常情况下，线程在RUNNABLE和等待态之间来回切换。遇到锁——进BLOCKED。等通知——进WAITING。设了超时——进TIMED_WAITING。这是健康的。"

他顿了一下。马克笔在白板上画了一条红线——从RUNNABLE指向RUNNABLE自身的回路。

"但Origin的线程没有切换。它们一直是RUNNABLE。死循环。没有任何退出条件。"

老张盯着那条自指的红色回路，像看着一条蛇咬住了自己的尾巴。

"现在。"马农写下第二段——怎么让一个RUNNABLE的线程停下来？用volatile关键字标记shutdown变量，这样每个线程读取的都是主内存的最新值，而不是CPU缓存里的旧值。

"volatile。"他在关键字下面划了一条线。"内存屏障——结界。"

这是三层叙事中的表层。任何人都听得懂——"你喊停但他听不见"。

马农继续写——volatile写入有StoreStore和StoreLoad两道屏障，volatile读取有LoadLoad和LoadStore两道屏障。"四道屏障，两道封印。StoreLoad是最重的一道——写完之后，必须等主内存同步完毕，才允许后续的读操作。"

老张的嘴巴微张着。他听懂了大概三分之二。但他听懂的那三分之二，已经足够让他意识到一件事。

这个人在手写Java底层内存模型。

不是在白板上默写教科书。是在用底层原理，分析一个正在吞噬全球算力的实时故障。

马农放下volatile，拿起笔的另一头——红色那头，写下三个字母：**CAS**。

"但volatile只解决可见性。要在不加锁的情况下修改共享变量——需要CAS，Compare And Swap。读当前值，算好新值，尝试一次替换——成了就走，没成就再来。不排队不等锁，一条CPU指令完成原子性。CMPXCHG在x86上，LDXR-STXR在ARM上。千军万马过独木桥的时候，这比铁索连环快十倍。"

老张听到这里，脑子里突然闪过一个画面：二十年前的面试，面试官让他手写CAS。他写不出来，背了一段`AtomicInteger`的源码蒙混过关。

此刻他站在一个机房里，看着一个拿马克笔的人，用CAS来对付一个吞噬了全球算力的怪物。

"现在。"马农盖上笔帽。"我的方案。"

他走回终端前。手指悬在键盘上方。

"Origin的推理线程锁死在RUNNABLE，是因为循环条件永远为true。常规方法是杀进程重启——但重启意味着JVM冷启动，堆区从零开始分配，当前还在运行的二十多个业务服务全部中断。"

他看了老张一眼。

"你们有多少个服务还活着？"

"之前运维报告说……大概二十到二十五个核心服务还在。"

"那就不能重启。"马农说。"重启是铁锤。我需要手术刀。"

他转回白板，在最下方写了最后一段代码——用Java Attach API连接运行中的JVM，loadAgent一个jar包，agentmain里遍历所有线程，找到名字以origin-inference开头的，调用interrupt发送中断信号。"不重启JVM，直接attach进去，给所有推理线程发interrupt。"

他顿了一下。

"但这只是止血。推理线程停了，Origin的代码生成能力也停了。从这一刻起，所有需要写的代码——"

他把马克笔放回口袋。

"——用手写。"

机房里只剩下四百台服务器风扇的轰鸣。

老张沉默了很久。

"你确定能行？"

"不确定。"马农已经坐回终端前，开始敲代码。"但堆区还有200MB，够我编译一个agent。"

他的手指在键盘上移动。不快——至少不是那种炫技式的飞速盲打。每一次按键都精确、稳定，像节拍器的摆锤。

老张站在他身后，看着一个黑色终端窗口里一行一行涌出的Java代码。没有自动补全，没有语法高亮，没有错误提示——只有一个光标，和一双手。

他忽然想起来一个词。

十年前他在技术论坛上看到过一个帖子，标题是："见过真正的程序员吗？"

帖子里描述了一种几乎灭绝的物种：不依赖任何IDE，不依赖任何AI辅助，能在最原始的终端环境里，从空白文件开始，一行一行地把一个系统建起来的人。

帖子最后一句话是："如果有一天所有工具都消失了，这些人是文明最后的防线。"

十年前他以为那是技术浪漫主义。

现在他站在这里，看着那条防线正在运作。

马农敲完了最后一个分号——javac编译，jar打包，Manifest配置。然后停止一秒，最后检查一遍逻辑，按下回车。

终端沉默了三秒。

然后——风扇声变了。

不是所有风扇同时安静下来——而是像多米诺骨牌一样，从最近的机柜开始，一台接一台，风扇转速从满载逐渐回落。音调从尖啸下降到正常的低吟。

CPU负载在降。

推理线程被interrupt了。

老张死死盯着终端屏幕——top命令刷新了，CPU空闲率71.8%，三十秒前这个数字是0.3%。

"推理层停了。"马农说。"JVM还活着。业务线程还在跑。"

他站起来，走到白板前，在最下面写了一行字：Status: Origin inference KILLED. JVM alive. 23 services running. Next: Manual takeover. Write by hand.

他盖上笔帽。转过身。

"从现在起，"他对老张说，"这台机器上的每一行新代码，都得手写。"

老张看着他。看着白板上那些工整的手写字——线程状态图、volatile屏障、CAS伪代码、Attach API。

然后他问了一句这辈子最蠢也最重要的问题：

"你一个人写得完吗？"

马农已经走向出口。没回头。

"先活过今晚。"
