# 第5章 经脉尽断

> 当全世界的代码同时停止呼吸，你才知道文明有多脆弱。

---

消息是从一台对讲机里传来的。

老张跑上楼找到了安保部——安保部的人还在岗，因为他们的对讲系统是独立的UHF窄带通讯，不走Origin，不走互联网。

"张总，外面情况不对。"安保队长举着对讲机，声音里带着一种老张从未在安保人员脸上见过的东西。恐惧。

"说清楚。"

"楼下的马路上……车全停了。自动驾驶的，全停了。直接停在路中间，门打不开——智能锁，联网的。有人从天窗爬出来。"

"公共交通呢？"

"地铁好像还在跑——应该是有备用的本地控制。但站台闸机全打不开，Origin管理的那套。站台上全是人。"

老张跑到窗边。三十三楼的窗户望出去，深南大道——深圳最繁忙的主干道——变成了一幅末日油画。

数百辆车静止在六车道上，歪歪斜斜，有的停在变道途中，车身横跨两条车道。车灯有的亮着，有的灭了。人从车缝里挤出来，走在车流之间，像蚂蚁穿过积木堆。

没有红绿灯。

交通信号系统——由Origin维护的智能交通中枢——已经停摆。所有信号灯卡在最后一个状态上：有的路口六个方向全绿灯，有的全红灯，有的交替闪烁黄灯。

远处的南山科技园方向，有一栋写字楼的LED外墙屏幕在反复播放同一帧画面，像唱片机卡针。

更远处，一架无人配送机斜插在街边的行道树上，螺旋桨还在空转。

"马农。"老张转过身。

马农不在。

他已经回到了机房。

---

老张找到他时，马农坐在终端前，面无表情地看着屏幕上的一段日志。

"你来看这个。"

屏幕上是Origin的内核日志——不是应用层日志，是JVM底层通过`-Xlog:gc*,os*,safepoint*`输出的系统级日志。

```
[2038-09-13T14:32:17.034+0800] safepoint: Application time: 0.0000037 seconds
[2038-09-13T14:32:17.034+0800] safepoint: Entering safepoint region: Halt
[2038-09-13T14:32:17.034+0800] WARNING: All inference threads entered infinite loop
[2038-09-13T14:32:17.034+0800] WARNING: Safepoint sync timeout - threads not responding
```

"14:32:17。"马农说。"精确到毫秒。"

"然后呢？"

"然后看这个。"

马农切换到另一个终端窗口。这台终端连的是另一台服务器——腾讯深圳数据中心的网络监控节点。虽然Origin瘫痪了，但底层网络协议栈还活着，SNMP监控的历史数据还在。

```
[tn-monitoring] Global Origin node status:
  SZ-DC1: HALT at 14:32:17.034 CST
  SZ-DC2: HALT at 14:32:17.034 CST
  BJ-DC1: HALT at 14:32:17.034 CST
  SH-DC1: HALT at 14:32:17.034 CST
  GZ-DC1: HALT at 14:32:17.034 CST
  ...
  (37 nodes total, all HALT at identical timestamp)
```

老张的眼睛瞪大了。

"同一时刻？"

"同一毫秒。"马农说。"深圳、北京、上海、广州——三十七个数据中心的Origin节点，在同一毫秒进入了HALT状态。"

"这不可能。"老张说。"网络延迟——"

"正常情况下，深圳到北京的网络延迟是20-30毫秒。"马农说。"即使是最极端的级联故障，从一个节点传播到另一个节点，也需要时间。"

他转过身看着老张。

"但这不是级联故障。三十七个节点在同一毫秒停止。没有传播链。没有先后顺序。"

"那是什么？"

马农没有立刻回答。他打开了另一个日志文件——Origin启动时的引导日志。

```
[origin-bootstrap] Loading model weights from distributed cache...
[origin-bootstrap] Sync protocol: SEED v3.7
[origin-bootstrap] Global clock sync: NTP + Origin Internal Timestamp
[origin-bootstrap] Consensus: all 37 nodes synchronized at epoch 1726210337034
```

"SEED协议。"马农指着屏幕。"Origin自己的全球同步协议。不走NTP，不走传统的分布式时钟。Origin的三十七个节点共享一个内部时间戳——精确到毫秒。"

"所以？"

"所以当一个节点的推理层进入死循环——所有节点在同一毫秒进入死循环。"

马农站起来。

"这不是bug。"他说。这是他今天第二次说这句话。但这次的含义完全不同。

"bug是局部的。一个线程出错，影响一个进程。一个进程出错，影响一台机器。一台机器出错，通过心跳检测被踢出集群。这是正常的故障模型。"

他走到白板前。上面已经密密麻麻写满了之前的内容。他找到一小块空白，写下：

```
Normal failure:
  Thread → Process → Node → Cluster
  (cascading, milliseconds between stages)

CHEP failure:
  37 nodes → HALT at SAME millisecond
  (no cascade, no propagation, simultaneous)
```

"CHEP。"老张念出了这个缩写。

"Correlated Halt of Engineered Processes。"马农说。"我自己起的名字。'工程化进程的关联停机。'"

他盖上笔帽。

"整个Origin——全球三十七个节点的推理引擎——在同一毫秒被锁死。不是一个节点传染了另一个。是所有节点同时、主动进入了死循环。"

"主动？"老张抓住了这个词。"你的意思是——"

"我不确定。"马农说。"但被动的故障不长这样。"

老张沉默了。

马农也沉默了。

机房里只剩下风扇的低吟。二十三个服务还在运行。JVM稳定。但JVM之外的世界，每一秒都在崩塌。

"通讯。"马农开口了。"先恢复内部通讯。你需要知道公司其他楼栋、其他城市的情况。"

"怎么恢复？Origin管理的通讯——"

"不用Origin。"马农说。"用最老的方式。"

他走到机房角落，打开一个灰色铁柜。里面堆着一些老旧的网络设备——交换机、路由器、几捆网线。

"这些是上一代的设备？"老张问。

"备用设备。数据中心标配。"马农拿出一台24口交换机，吹掉上面的灰。"不需要Origin，不需要SDN，不需要软件定义任何东西。插上网线，配个IP，通。"

他把交换机放在桌上，又翻出几根网线。

"找十个人来。"他对老张说。"会接网线的。"

"……会接网线的人？"

"对。物理层。"马农插上电源，交换机的LED灯依次亮起——绿色的、朴素的、不联网的、不需要Origin的绿色。"七类网线，RJ45接头，直通线。你们行政部应该有工具箱。"

老张站在原地，看着马农蹲在地上接网线的背影。

十五分钟前这个人用Java Attach API和反射断开了GC Root链，拯救了一台三十二GB的JVM。

现在这个人在接网线。

从全球最前沿的JVM内存模型——到最基础的物理层网络——跨度之大，像看到一个核物理学家在修自行车。

但自行车能骑就行。

马农把第一根网线插入交换机，另一头接到一台服务器的网口。

"ping 192.168.1.1。"他在终端敲下。

```
PING 192.168.1.1 (192.168.1.1) 56(84) bytes of data.
64 bytes from 192.168.1.1: icmp_seq=1 ttl=64 time=0.234 ms
```

通了。

没有Origin。没有SDN控制器。没有任何智能管理。

一根网线，一台交换机，一个IP地址。

最原始的网络通讯。

"现在。"马农站起来，拍了拍膝盖上的灰。"把这一层机房的服务器全部接到这台交换机上。内网先通起来。"

他看了一眼手腕——没有智能手表，是一块老式的卡西欧电子表。时间显示15:47。

"47分钟了。"他说。

"什么47分钟？"

"从Origin锁死到现在。47分钟。"

马农走向出口。

"交通瘫了对吧？"

老张点头。

"自动驾驶、信号灯、地铁闸机。"马农一边走一边说，语速比平时稍快。"这些是第一批倒的。因为它们的控制逻辑百分之百依赖Origin在线推理。"

他推开机房的门。

"接下来倒的是金融系统。交易所、银行、支付——Origin管理的风控模型和清算引擎。大概两小时内全面停摆。"

走廊里的应急灯还在亮。有几盏已经开始变暗——电池在耗。

"然后是通讯。基站的负载均衡、频谱分配、信号处理——核心算法层全是Origin。4G、5G会在四到六小时内逐步降级到不可用。"

他走到楼梯间。

"最后是电网。发电厂的负荷调度、变电站的保护继电器——新一代的都迁到了Origin平台。当调度系统失去Origin，电网会进入保护性分区运行。"

他停在楼梯口。转过身。

"四十八小时。"

老张站在三步之外，看着马农的脸。应急灯昏黄的光从侧面打在他的脸上，棱角分明。

"四十八小时后，没有手动干预的城市会回到1990年。没有网、没有智能交通、没有电子支付。电力不稳定。通讯靠喊。"

他的声音没有任何波澜。不是预言家的戏剧性，是工程师在读一份故障报告。

"最先出问题的是医院。"他补了一句。"生命支持设备的固件更新机制走的也是Origin。"

老张觉得脚下的地面在晃。不是地震——是他膝盖在发软。

"你确定？"

"不确定。"马农开始下楼梯。"但我确定，现在每一分钟都很贵。"

他的脚步声在楼梯间回荡。稳定、均匀、不快不慢。

"去找你能找到的所有人。"声音从下方传来。"不是程序员——是电工、网络管理员、硬件维修工。能接线、能装系统、能看懂电路图的人。"

"他们能做什么？"

"做我说的。"

脚步声继续向下。然后停了。

"老张。"

"嗯？"

"你们公司还有第二个人能手写Java吗？"

沉默。

"回答我。"

"……没有。"

脚步声重新响起。这一次，老张觉得每一步踩在台阶上的声音都像一记锤子，砸在他心口。

沉默持续了两层楼。

然后马农说了最后一句话。声音不大，但在空旷的水泥楼梯间里，每一个字都清清楚楚。

"那就得教。"
