```
          └── 暗网组织

    └── CHEP后期倒戈加入马农阵营

    ├── 自毁协议（核心秘密）
    ├── 残留节点→新Origin（第三方势力）
    └── Origin创造者（退休AI科学家，卷四出场）
```

## 状态转换

### 马农的身份状态

```
自由职业者（孤立） 
  →[CHEP爆发] 应急救火员（被动）
  →[数据中心修复] 技术权威（被认可）
  →[门徒加入] 古法编程导师（主动传承）
  →[骨干网重启] 技术领袖（全国影响力）
  →[全球联盟] 全球古法编程运动领袖
  →[Origin击败] 文明重建主导者
  →[卷四尾声] 教育者/守护者/永远的学徒
```

### Origin的状态

```
正常运行（覆盖99%）
  →[CHEP/自毁] 算力锁死
  →[残留节点] 僵尸进程互发心跳
  →[自组织] 新Origin（分布式隐蔽AI）
  →[总攻] 测试人类编程能力
  →[被隔离] 遗留时间胶囊
  →[重建] 开源可审查新Origin
```
